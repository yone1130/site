よね/Yone Discordサーバーガイドライン

よね/Yone のサーバーに参加いただきありがとうございます。
当サーバーを利用するにあたって、本ガイドラインを守って、利用して頂く必要があります。
よくお読みください。

1. 定義

  1. 「当方」

    本サーバーの設立・運営を行っている、よね/Yoneを指します。

  2. 「メンバー」

    本サーバーに参加されているすべての方を指します。

  3. 「他のメンバー」

    本サーバーに参加されているあなた以外のすべての方を指します。

  4. 「第三者」

    メンバー以外の方を指します。

  5. 「個人情報」

    個人を特定することができる情報を指します。また、それ単体では特定できないものでも、他の情報と組み合わせることにより特定可能になりうる情報を指します。

2. 本ガイドラインの適用範囲

  1. メンバーは、本サーバーを利用するには本ガイドラインに同意しなければなりません。
  2. メンバーは、本サーバーを利用した時点で本ガイドラインが適用されるものとします。
  3. メンバーは、本ガイドラインの定めにしたがって本サーバーを利用するものとします。

3. 参加要件

メンバーは、本サーバーを利用するにあたり、以下の要件を満たす必要があります。

  1. 年齢が満13歳以上であること

要件が満たせない場合には、速やかに本サーバーから脱退いただくようお願いいたします。
これらを偽って本サーバーに参加したことが判明した場合はキックもしくはBANをさせていただきますのでご了承下さい。

4. 禁止事項

本サーバー内において、以下の行為を禁止とします。
以下の行為を発見した場合、当方の判断に基づきタイムアウト、BAN等の処置をさせていただきます。
また、本サーバー外（他サーバーや他のSNSプラットフォーム）において同様の行為を発見した場合に、当方の判断に基づき処置をさせていただく場合がございます。

  1. あなたの居住している国もしくは地域で定められている法令に反する行為
  2. 裁判所の判決もしくは命令，または法令上拘束力のある行政措置に反する行為
  3. Discord利用規約に違反する行為
  4. 当方、他のメンバーおよび第三者の著作権、商標権、特許権等の知的財産権、名誉権、プライバシー権、肖像権、人格権、またはその他法令上または契約上の権利を侵害する行為
  5. 公序良俗に反する行為
  6. 個人情報を含む内容を発言する行為
  7. 差別的な内容を発言する行為
  8. 暴言、中傷、脅迫または不快、侮辱的な発言をする行為
  9. 露骨な性的コンテンツ（画像、動画、イラスト）を発言する行為
  10. 露骨な性描写を言語で表現し、発言する行為
  11. 当方、他のメンバーおよび第三者になりすます行為
  12. 自殺、自傷行為、薬物乱用を誘引または助長する行為
  13. 宗教活動または宗教団体への勧誘行為
  14. 連投、スパム行為
  15. 本サーバー及びその他のサーバーに支障を与える行為
  16. 根拠のない物事を確定情報のように発言する行為
  17. @everyone及び@here を @管理者 の許可なく使用する行為
  18. 次のようなユーザー名およびニックネームは使用してはいけません
    • 「everyone」および「here」
    • 卑猥または不快なもの
    • 差別的または中傷的な文章が含まれるもの
  19. 次のようなユーザーアイコンは使用してはいけません
    • 卑猥または不快なもの
  20. その他当方が不適切と判断した行為

5. 利用規則

本サーバー内において、以下の規則を厳守してください。
以下の規則に反する行為を発見した場合、当方の判断に基づきタイムアウト、BAN等の処置をさせていただきます。

  1. URLの送信について

    テキストチャンネルでは、URLを送信することが可能ですが
    過度な送信は個人チャンネルで行ってください。
    なお、詐欺サイト、違法サイト等はスパムとして扱われます。

  2. NSFWチャンネルについて

    NSFWチャンネルでは、性的コンテンツの発言が一部許可されています。

    ただし以下のコンテンツは発言してはいけません。

    • 実写による性的コンテンツで、性器等が含まれるもの
    • 児童ポルノに該当するもの

  3. 実行ファイルについて

    コンピュータで実行可能なファイルを直接送信しないでください。
    配布をする場合は GitHub Release や Google Drive 等を用いて行ってください。

    実行ファイルとは次のようなものを指します。

    • .exe、.msi - Windows実行ファイル
    • .apk - Androidアプリケーションパッケージ
    • .app - MacOS実行ファイル
    • .ipa - iOSアプリケーションアーカイブ

7. 本ガイドラインの変更

  1. 当方は、メンバーに個別の同意を得ることなく、本ガイドラインの内容を変更することができるものとします。
