出典

災害情報を投稿するに際し、次の機関から情報を提供させていただいています。

  • 国立研究開発法人防災科学技術研究所 の提供する 強震モニタ を利用して、
    国土交通省・気象庁 が発表する緊急地震速報を取得。
  • P2P地震情報 が提供する JSON API v2 を利用して、
    国土交通省・気象庁 が発表する地震情報を取得。
    この情報の利用に際し、気象庁ホームページの利用規約 が適用されています。
これらの情報を提供してくださる方々に感謝申し上げます。
